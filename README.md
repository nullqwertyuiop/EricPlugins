## 注意

直接 fork 现有的插件仓库可能会使 Eric 出现插件解析失败，请直接使用模板仓库创建

## 开始使用

### 插件托管

本仓库使用 `Github Actions` 对插件进行处理，在 `main` 分支更新插件后，`Actions` 将会自动将更新后的插件推送至 `modules` 分支

### 安装插件

可通过 Eric 内置的 `管理器` 进行插件安装，也可自行从 `modules` 分支下载所需插件进行安装
